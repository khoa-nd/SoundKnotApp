🟡 Thumbnail letterbox (16:9 source in square/portrait slots)

320×180 thumbnails cover-fit into 160×160 (Recent) and 150×200 (Trending) boxes, leaving bars — reads as unintentional framing.

FixCrop-fill edge-to-edge (zoom, no bars) or change the thumbnail boxes to 16:9. (Open decision.)

🟢 Marker glyph vs SVG icon — the ▶ YouTube marker is a text glyph, not an SVG from the icon family. Pro Max no-emoji-icons / icon-style-consistent prefer a vector. Swap ▶ for the same Lucide/Ionicons play/youtube icon used elsewhere.
